# c++的模运算

```cpp
c++ 规定 / 向零取整导致余数可能是负数
assert(5 % 3 == 2);  // 5 / 3 == 1 ... 2
assert(5 % -3 == 2); // 5 / -3 = -1 ... 2
assert(-5 % 3 == -2); // -5 / 3 = -1 ... -2
assert(-5 % -3 == -2); // -5 / -3 = 1 ... -2
但是模严格来说是正数, 所以我们在余数为负数的时候应该加上除数来使得模为正数
```
